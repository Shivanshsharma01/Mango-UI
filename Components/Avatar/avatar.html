Avatar
Avatars are used to show profile images in headers or profile information sections. They can be in multiple sizes.
Round Avatar with Image
Use class avatar followed by size to create circular alert with image.
Square Avatar with Image
Use class alert followed by square and size respectively to create square avatar with image.
